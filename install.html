How to install the custom block site extension

How to install the block site extension for use in a few clicks

1.Download the extension files here

2. Visit the chrome extension page by clicking on the triple vertical dots icon on the top right of the browser, followed by selecting more tools, then clicking on the extensions option

3. Enable developer mode and select Load unpacked. Navigate to the where the downloaded extension file is located and select the downloaded folder.

4. Once loaded, copy the extension ID and input it below. Submit and you are all set!
